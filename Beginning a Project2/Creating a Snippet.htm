Creating a Snippet

 

A snippet is for any content that you want to use more than once. If you are designing a user guide and want to include the title in the header of each page, you would make the title into a snippet. It is like a mini topic, but it is saved in a separate file so that you can insert it into your project wherever you want to and as many times as you want. A snippet can be a single line or a paragraph, and can include images as well.

 

Creating a snippet from text that you already have:

 

  1. Select the text that you want to turn into a snippet.

 

  1. Go to the Home tab.

 

  1. Click on Create Snippet. The Create Snippet dialogue box appears.

 

  1. Under Snippet File, choose a name for your snippet. Leave the Project Folder as Resources/Snippets. Under Snippet, you can see the text that you selected.

 

  1. Make sure that the Replace Source Content with the New Snippet box is selected.

 

  1. Click Create. Your selected text now has brackets around it, showing that it is a snippet. If you go to the Content Explorer and click on Resources>Snippets, you can see the new snippet file that you have created.

 

Creating a snippet from scratch:

 

  1. Right click in the Content Explorer.

 

  1. Go to New>Snippet. The Add File dialogue box appears.

 

  1. Under File Type, make sure that Snippet is selected.

 

  1. Under New From Template, Choose Factory Templates>Snippet.

 

  1. Under File Name, choose a name for your snippet file.

 

  1. Click Add. The snippet file automatically opens in the main window.

 

  1. Delete the default text and add your own. Make sure to save afterwards.
