Creating a Variable

  1. Open the Project Organizer window.

 

  1. Double click the Variables folder and choose a Variables Set. If you don’t have a Variables set, create one by doing the following:
  1. Click on the File Menu
  2. Click on New. The Add File window appears.
  3. Change the drop down menu at the top to Variable Set. Name your new variable set.
  4. Click Next. A new variables set appears in the Variables Folder. If you didn’t name your Variables set in the Add File window, it will have the generic name New Set. Click on it to rename it here in the Variables Folder.

 

  1. Double click on the new variables set. The VariableSet Editor opens.

 

  1. Click on the Add Variables icon in the VariableSet Editor toolbar. A new variables line is added to the VariableSet Editor.

 

  1. Double click in the appropriate field to enter a new name or new definition for the new variable.
