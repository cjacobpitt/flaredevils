Creating a New Topic

Flare-Devils get ready! You are about to embark on a new journey of topic based authoring. Follow the steps below to create your very first new topic.

  1. Open the Content Explorer Window.
  2. Select the Creating a New Topic icon in the Content Explorer toolbar.
  3. Choose a topic type from the Factory Templates. The main window appears and contains an h1 heading and some paragraph content that can be replaced with your own content.
  4. Note that you are currently in the XML Editor view. Click the Text Editor tab to change to an HTML 5 view.
  5. Select the tiny inverted v in the lower right corner of either the XML or Text Editor view to view both views side by side.
