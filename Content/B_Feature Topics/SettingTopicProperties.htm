Setting Topic Properties

Each topic that you make in MadCap Flare will fit into the document differently. Some topics will only appear in the print versions of your guide, some will only appear in the online version, and some will appear in both. Additionally, each topic will have a stylesheet attached to it, which will inform what the topic page looks like.

 

Editing Topic Properties

 

  1. To look at and edit the properties of a topic page, right-click on the topic in the Content Explorer. A dropdown menu opens.
  2. Click on the Properties option. A Properties window appears. Alternatively, you can simply select the topic in the Content Explorer and press the F4 key on your keyboard.

 

To Set the Conditions

 

  1. In the Properties window, click on Snippet Conditions in the sidebar.
  2. In Snippet Conditions, there is a table with each set of condition tags. Inside these sets, by default there are PDF, Online, and XHTML tags. Each tag has an Include and Exclude option.
  3. By choosing to Include, the topic will be included in the chosen output.
  4. Conversely, by choosing Exclude, the topic will be excluded from the chosen output.
  5. Once the condition tags have been set, click OK to close the Properties window.

 

To Set the Stylesheet

 

  1. In the Properties window, click on Topic Properties in the sidebar.
  2. In Topic Properties, click on the Stylesheet option. A dropdown menu opens.
  3. In the dropdown menu, choose the desired stylesheet option.
  4. Click OK to close the Properties window.
