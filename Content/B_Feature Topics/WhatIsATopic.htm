What Is A Topic?

The term Topic refers to any chunk of information that is contained within your Flare manual. Much like chapters or modules, topics are used to separate content into digestible and easily organized sections. Topics are the main content of any manual, and Flare makes every effort to make topics easy to navigate and edit.

 

Navigating Topics

 

All topics can be found in the Content Explorer, which is the sidebar on the left part of the screen. That sidebar contain both the Content Explorer and the Project Organizer. If the sidebar is set to show the Project Explorer, click on the button that says Content Explorer located at the bottom of the sidebar.

 

Inside the Content Explorer is a series of folders. If you are using a template provided by Flare, then there will be multiple folders for different topic types. At the moment, these are placeholder files with no text. Each topic will be in the .htm format, to allow it to be formatted for web and print. Topics will almost always be edited by the user, but there are some cases where Flare can generate content for a topic, such as when creating a table of contents, glossary, or index.

 

Fill Up: Content is edited within either the Text Editor or the XML Editor. The Text Editor is designed to be more like InDesign or Microsoft Word, with a focus on how the finished product will look. The XML Editor is focused more on the organization of the writing itself, containing the styles and text tags that will show up in the web version.
