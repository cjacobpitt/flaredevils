Working with Styles

MadCap Flare offers support for all kinds of styling over a variety of outputs. This means that any styling you use for a PDF guide can be easily translated into an HTML guide. The way this is done is by having styles be formatted as CSS (or Cascading Style Sheets). As a result of this, working with styles is a bit more involved than it would be if you were working with InDesign or Microsoft Word.

 

Creating a New Stylesheet

 

  1. There are multiple ways to create a new stylesheet:
    1. Click on the folder where you would like your stylesheet to be stored. Click on the Project tab in the Ribbon. In the Content section, click on the New icon. A dropdown menu opens. In the dropdown menu, click Add Stylesheet. The Add File window appears.
    2. Right-click on the folder where you would like your stylesheet to be stored. A right-click menu appears. Hover over the New section. Another menu appears. Click on Stylesheet. The Add File window appears.
  2. In the Add File window you may choose to either create your stylesheet from a template by choosing from the New from template box, or duplicate an existing stylesheet by choosing a stylesheet from the New from existing box.
  3. Choose a name for your stylesheet.
  4. Click Add.
  5. Your new stylesheet appears in the chosen folder.

 

Editing a Stylesheet

 

  1. Double-click on the stylesheet that you would like to edit. The stylesheet properties appear in the main window.
  2. There are two ways to view the stylesheet, the simplified and advanced views. The simplified view is the default view and it shows all of the style classes as a large table. The advanced view shows the style classes as a list of dropdown menus. Near the top of the main content window is the Stylesheet Editor menu. Click on View: Advanced or View: Simplified to change the view.
  3. In the simplified view, to edit a style, double-click on the style class you would like to edit. The Properties window appears. In the Properties window you can edit fonts, sizes, layouts, backgrounds, and much more.
  4. Click OK to close the window.
  5. In the advanced view, to edit a style, click on the style to access its properties in the main content window. The properties are organized into font, box, block, etc. Each of these categories contain the specific CSS elements that can be edited.
  6. Save to implement your CSS changes.

 

Attaching Styles to a Topic

 

  1. In the Content Explorer, right-click on the topic you want to attach a stylesheet to. A right-click menu opens.
  2. Click on the Properties in the right-click menu. The Properties window appears.
  3. In the Topic Properties section, choose a stylesheet in the Stylesheet section by clicking on the Select button. The Stylesheet Links window appears.
  4. On the left are all of the available stylesheets to add to the topic, and on the right are the stylesheets that have been added to the topic. To add or remove a stylesheet, click on the Left or Right Arrows (Add button image).
  5. Click on the Add File button to create a new stylesheet and add it to the topic.
  6. Click OK when you are done.

 

Master Styles

 

  1. To add master styles to an output:
  2. In the Project Organizer, double-click on the target that you would like to add a master stylesheet to. That target’s properties open in the main window.
  3. In the target’s properties window, click on the General tab if you are not already there.
  4. In the Master Stylesheet dropdown menu, you are presented with a list of all of the project’s stylesheets. Choose one and click on it.
  5. To make changes to the master stylesheet, you can follow the steps outlines above in Editing a Stylesheet, or you can click Edit next to the Master Stylesheet dropdown menu.
  6. Save your changes.
