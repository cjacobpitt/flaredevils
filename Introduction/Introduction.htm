Introduction

 

This manual will put beginner to intermediate level technical writers on the road to creating successful single-source documentation. You will be guided through the creation of your own project from start to finish, and will learn the ins and outs of Flare along the way.

 

You will learn topics such as:

 

  • How to add topics
  • How to use snippets and variables
  • How to format your project with CSS
  • How to use targets to create outputs

 

Madcap Flare is a modern, useful tool for technical writers that will save you a lot of time and energy. This guide will help you to understand the basics, so that you will be on your way to becoming a seasoned technical writer.
